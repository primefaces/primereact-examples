# PrimeReact + Headless

A minimal [PrimeReact](https://primereact.dev) v11 starter for Next.js using **headless** mode — unstyled hooks you bring your own markup and styling to, with class-based (`.dark`) dark mode and the Inter font.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and edit `app/page.tsx` to start.
