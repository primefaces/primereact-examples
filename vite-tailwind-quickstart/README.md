# PrimeReact + Vite

A minimal [PrimeReact](https://primereact.dev) v11 starter for Vite using **Tailwind** mode — PrimeReact primitives wrapped with Tailwind + PrimeUI tokens, with class-based (`.dark`) dark mode and the Inter font.

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and edit `src/App.tsx` to start.
