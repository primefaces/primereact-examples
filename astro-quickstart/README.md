# PrimeReact + Astro

A minimal [PrimeReact](https://primereact.dev) v11 starter for Astro using **styled** mode — pre-themed components driven by design tokens, with class-based (`.dark`) dark mode and the Inter font.

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) and edit `src/components/PrimeApp.tsx` to start.
