# PrimeReact + Tailwind

A minimal [PrimeReact](https://primereact.dev) v11 starter for Next.js using **Tailwind** mode — PrimeReact primitives wrapped with Tailwind + PrimeUI tokens, with class-based (`.dark`) dark mode and the Inter font.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and edit `app/page.tsx` to start.
