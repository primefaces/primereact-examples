# PrimeReact + TanStack

A minimal [PrimeReact](https://primereact.dev) v11 starter for TanStack using **Tailwind** mode — PrimeReact primitives wrapped with Tailwind + PrimeUI tokens, with class-based (`.dark`) dark mode and the Inter font.

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and edit `src/routes/index.tsx` to start.
